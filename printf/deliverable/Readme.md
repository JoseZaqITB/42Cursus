# Description

it is C library with a set of functions that will be useful for the 42 cursus. Most of the functions are a copy of the glibc. Others are very interesting functions. The purpose is to have a very useful library for future project in C.

If you want to check it out. You can start in the list below where you can find the function name ( and its link ) and a short description.

# Instructions

In order to use it you should first create the library with

## Create library
```
make
```
This will create all the .o files and the library libft.a
Then you can compile your main file

if you want to ged rid of the .o files
```
make clean
```
if you also wnat to remove the .a file
```
make fclean
```

and for a remake
```
make re
```

## execute the main.c
```
ccc main.c -L. -lft -I.
```
